Yêu cầu bài tập

Vận dụng các kiến thức về thẻ h1 và p để tạo 1 trang web tương tự dưới đây

Kiến thức bổ sung: Có 2 cách để đưa hình ảnh vào web của bạn

  • Chèn ảnh bằng chính đường link lấy từ 1 web nào đó(Chú ý: nếu chủ sở hữu của hình ảnh đó thay đổi đường dẫn thì ảnh của bạn sẽ bị sai đường dẫn)
  • Chèn ảnh từ máy của bạn: đưa hình ảnh vào project của bạn và gọi nó ra bằng đường dẫn tương đối/tuyệt đối
    • Đường dẫn tương đối: ./ để gọi file folder cùng cấp, ../ để quay lại 1 cấp
    • Đường dẫn tuyệt đối: /
