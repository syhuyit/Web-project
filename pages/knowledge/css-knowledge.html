Kiến Thức Cơ Bản CSS

1. Tổng quan về CSS

CSS (Cascading Style Sheets) là ngôn ngữ dùng để mô tả cách trình bày của tài liệu HTML. CSS giúp bạn kiểm soát màu sắc, phông chữ, khoảng cách, bố cục và nhiều khía cạnh khác của trang web, giúp nó trở nên hấp dẫn và dễ sử dụng hơn.

2. Cú pháp CSS

Cú pháp CSS gồm các selector và declaration. Selector là phần dùng để chọn phần tử HTML cần định dạng, còn declaration là phần mô tả các thuộc tính và giá trị của chúng.

2.1. Selector

Selector là phần dùng để chọn phần tử HTML cần định dạng. Có nhiều loại selector khác nhau như selector lớp (class selector), selector id (id selector), và selector phần tử (element selector).

2.2. Declaration

Declaration là phần mô tả các thuộc tính và giá trị của chúng. Mỗi declaration bao gồm một thuộc tính (property) và một giá trị (value), được phân cách bởi dấu hai chấm (:). Các declaration được đặt trong dấu ngoặc nhọn ({}) và cách nhau bằng dấu chấm phẩy (;).

3. Cấu trúc và Cách chèn CSS

Bộ chọn cơ bản:

  • Selector phần tử (Element Selector): Chọn tất cả các phần tử có tên cụ thể. Ví dụ: p { color: red; }
  • Selector lớp (Class Selector): Chọn tất cả các phần tử có một lớp cụ thể. Ví dụ: .my-class { font-size: 16px; }
  • Selector id (ID Selector): Chọn phần tử duy nhất có một ID cụ thể. Ví dụ: #my-id { background-color: blue; }

Cách chèn CSS vào trang web:

  • Inline CSS: Chèn trực tiếp vào thẻ HTML bằng thuộc tính style. Ví dụ: <h1 style="color: red;">Tiêu đề</h1>
  • Internal CSS: Chèn trong phần <head> của tài liệu HTML bằng thẻ <style>. Ví dụ: 
    <style>
  h1 {
    color: red;
  }
</style>

  • External CSS: Chèn bằng cách liên kết đến một tệp CSS bên ngoài bằng thẻ <link>. Ví dụ: 
    <link rel="stylesheet" href="styles.css">

4. CSS Box Model (Mô hình hộp)

Đây là kiến thức quan trọng nhất để dàn trang. Mọi phần tử HTML đều được coi là một cái hộp bao gồm 4 lớp từ trong ra ngoài:

  • Content (Nội dung): Phần chứa nội dung thực tế của phần tử.
  • Padding (Đệm): Khoảng cách giữa nội dung và viền của phần tử.
  • Border (Viền): Đường viền bao quanh phần tử.
  • Margin (Lề): Khoảng cách giữa phần tử và các phần tử khác xung quanh nó.
