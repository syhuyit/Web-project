Lập trình web căn bản

Tổng quan về lập trình web

Ở khóa học này chúng ta sẽ tìm hiểu về 2 thứ đó là: HTML, CSS
Trong đó: HTML sẽ xây dựng bộ khung và text cho web, CSS sẽ đóng vai trò tô màu, căn chỉnh... cho web

Lưu ý: khóa học này sẽ lược bỏ khá nhiều lý thuyết, chỉ để lại các kiến thức cơ bản và quan trọng nhất nhằm mục đích đẩy nhanh quá trình học bộ môn Lập trình web

1. HTML basic

Thẻ Mô tả
h1-h6 Thẻ tiêu đề theo cấp độ từ lớn đến bé
p Xác định 1 đoạn văn bản trong html
br Dùng để ngắt dòng
hr Dùng để tạo ra 1 đường kẻ ngang
<!– –> Dùng để note, tương tự /* */ trong java
b In đậm
i In nghiêng
u Gạch chân
strong Dùng để đánh dấu các văn bản ở mức độ quan trọng, khi hiển thị trên màn hình thì như thẻ b
em Dùng để nhấn mạnh văn bản quan trọng, tương tự thẻ i khi hiển thị
pre Giữ nguyên định dạng văn bản mà bạn muốn thể hiện-giữ các khoảng trắng, xuống dòng
code Mang ý nghĩa câu lệnh trong lập trình
table Được dùng để tạo bảng
thead Xác định phần đầu của bảng
tbody Xác định phần thân của bảng
tfoot Xác định phần chân-nội dung tổng kết
tr Tạo 1 hàng trong bảng
th Phần tử con của tr, xác định 1 ô tiêu đề
td Phần tử con của tr, xác định 1 ô dữ liệu
